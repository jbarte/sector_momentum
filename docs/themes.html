How to read this tab

Themes are plotted by relative strength vs benchmark (x-axis) and whether that relative strength is accelerating or decelerating (y-axis). Both axes are centred at 100.

Four quadrants

  • Leading (top-right): Outperforming AND accelerating.
  • Weakening (bottom-right): Outperforming but decelerating. Prime exit / reduce signal.
  • Lagging (bottom-left): Underperforming AND decelerating. Avoid.
  • Improving (top-left): Underperforming but accelerating. Prime entry signal.

How to use

Themes rotate Leading → Weakening → Lagging → Improving → Leading. Buy in Improving before the move to Leading; sell in Weakening before the drop to Lagging. Dotted tail lines show the last scan positions — the direction of the tail matters more than the current position.

How to read this tab

Select a theme to see its score component history across all stored scans.

Key patterns

  • Composite rising + Change score rising: sustained acceleration. Highest-conviction entry setup.
  • Composite near peak + Change score turning negative: the theme is topping. Reduce or exit.
  • Composite falling but Change score rising: bottoming process. Watch for confirmation before entering.
How to read this tab

Rank change between the two most recent scans. Green bars = themes that climbed the leaderboard; terracotta bars = themes that slipped.

How to use

  • A single large positive move can be noise — confirm in the Drill-down tab before acting.
  • Themes with 2+ consecutive large positive moves in the History tab carry more conviction.
How to read this tab

Composite score trend across all stored scans for every theme. The composite is a cross-sectional z-score — 0 = theme-average for that scan.

Key patterns

  • Consistent upward slope: quality momentum sustained over multiple scans.
  • Sharp spike then reversal: likely a one-time event, not sustainable.
  • Gradual recovery from negative territory: potential emerging opportunity.

Monthly top-3 rotation, equal-weight, long-only. Price-based signals only (sentiment excluded). Themes scored within their own cohort vs ACWI.
